Released Nilvana Vision Inference Runtime 1.0.2

Release 1.0.2 of vision inference runtime, a user-friendly web-based model optimization, and application deployment No-code toolkit.

We are pleased to introduce version 1.0.2 of vision inference runtime, a user-friendly web-based model optimization, and application deployment No-code toolkit. You can find more information about Nilvana™ vision inference runtime on the official website.

Intuitive Software Updates

Simply launch your installer and click the check for updates button.

No command, No code for Inferencing

No need to write and modify model inference code, just enable the model API endpoint through the graphical interface settings to speed up model deployment time and start solving problems with the model in practice.

Easily Switch and Manage Vision Project Models

By enabling or disabling the model API endpoint, you can quickly switch between your AI applications or enable multiple AI inspection models at the same time.

Nilvana™ has developed Vision Studio and Vision Inference Runtime toolkits, which enable collaborative data annotation, model creation, and rapid inference without AI expertise.

Nilvana™ provides a wide range of services from consulting and model creation to the development of AI-based applications. Feel free to contact us for any inquiries.
