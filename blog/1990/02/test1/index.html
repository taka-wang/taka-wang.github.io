❌ Youtube 外掛字幕測試

1. 放在 Footer

<script>
  const sub = 'https://blog2.cmwang.net/subtitles/sub/astra.srt'; // change me
</script>
  
<script defer>
//--- don't change!
window.onload = function () {
  console.log('----Loaded----');
  loadSRT(sub, function (subtitles) {
    initSub(document.querySelector('.kg-card.kg-embed-card iframe'), subtitles);
  });
};
</script>

2. 如果想要更改透明度

<style>
  .youtube-external-subtitle span {background: rgba(0, 0, 0, 0.5);}
</style>
